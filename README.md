# bbs
这是一款基于B/S结构的校园BBS论坛

# 目录树

```
├─src
│  └─bbs
│      ├─biz
│      ├─dao
│      │  └─impl
│      ├─entity
│      ├─servlet
│      └─struts
│          ├─action
│          └─form
└─WebRoot
    ├─business
    ├─image
    │  └─heads
    ├─META-INF
    ├─style
    └─WEB-INF
        ├─classes
        │  └─bbs
        │      ├─biz
        │      ├─dao
        │      │  └─impl
        │      ├─entity
        │      ├─servlet
        │      └─struts
        │          ├─action
        │          └─form
        └─lib
```

# 上传项目代码
src存放java业务代码，WebRoot存放页面代码
